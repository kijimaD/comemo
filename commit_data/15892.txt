    runtime: improve mmap return value checking for netbsd/openbsd
    
    Rather than just checking for ENOMEM, check for a return value of less
    than 4096, so that we catch other errors such as EACCES and EINVAL.
    
    R=golang-dev, rsc
    CC=golang-dev
