    [dev.ssa] cmd/compile/ssa: handle loops that don't loop
    
    Loops such as
    
    func f(c chan int) int {
            for x := range c {
                    return x
            }
            return 0
    }
    
    don't loop. Remove the assumption that they must.
    
    Partly fixes the build.
