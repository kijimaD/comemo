    runtime: set HeapGoal to zero when the GC is disabled
    
    When the GC is disabled, the tracer should emit a heap goal of 0. Not
    setting the heap goal to 0 causes an inaccurate NextGC value to be
    emmited.
    
    Fixes #63864
