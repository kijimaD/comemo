    cmd/compile: Ensure left-to-right assignment
    
    Add temporaries to reorder the assignment for OAS2XXX nodes.
    This makes orderstmt(), rewrite
      a, b, c = ...
    as
      tmp1, tmp2, tmp3 = ...
      a, b, c = tmp1, tmp2, tmp3
    and
      a, ok = ...
    as
      t1, t2 = ...
      a  = t1
      ok = t2
    
    Fixes #13433.
