    all: separate doc comment from //go: directives
    
    A future change to gofmt will rewrite
    
            // Doc comment.
            //go:foo
    
    to
    
            // Doc comment.
            //
            //go:foo
    
    Apply that change preemptively to all comments (not necessarily just doc comments).
    
    For #51082.
